# 🎨 PUTwiki - Client (Frontend)

The PUTwiki client is the user interface layer of the application, built with React and designed for scalability, maintainability, and fast development.

## 🏗️ Architektura overview

**PLACEHOLDER**

## 🛠️ Tech Stack

- **Bun** – Package manager and runtime
- **Vite** – Fast development server and build tool
- **TypeScript** – Static typing
- **React** – UI library
- **Tailwind CSS** – Utility-first styling
- **shadcn/ui** – Accessible and customizable UI components
- **Prettier** – Code formatter

## 🚀 How to run (dev)?

```bash

cd client

bun install

bun run dev
```

## Notes

> ⚠️ Please note that you need to have bun installed to run this app.
